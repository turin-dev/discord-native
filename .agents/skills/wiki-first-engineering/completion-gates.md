# Completion Gates

## Start Gate

- 이번 작업이 코드 변경을 포함하는지 먼저 본다.
- 포함한다면 관련 위키 축을 고르고 `wiki-query`를 선행한다.
- 같은 스레드에서 이미 충분한 위키 문맥을 읽었다면 완전 재탐색은 생략하되, 관련 노트 확인 자체는 생략하지 않는다.

## Finish Gate

- 작업 종료 전 `wiki-sync`를 반드시 거친다.
- 결과는 아래 둘 중 하나여야 한다.
  - 위키를 갱신했다.
  - 갱신 불필요를 명시적으로 판단했다.
- 여러 문서, 허브, 새 구조를 건드렸다면 `wiki-lint`를 추가로 돌린다.
- 구조 문제가 남아 있으면 `wiki-repair` 또는 `wiki-expand`로 넘긴다.
